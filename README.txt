2. Puedes subirla a GitHub siguiendo estos pasos:

PASOS PARA PUBLICAR EN GITHUB PAGES:
-----------------------------------
1. Crea una cuenta en https://github.com (si no tienes una).
2. Crea un nuevo repositorio, por ejemplo: harmonie-web.
3. Sube los archivos de esta carpeta (index.html, style.css y /img).
4. En GitHub, ve a "Settings" > "Pages".
5. En la sección "Source", selecciona "Deploy from a branch".
6. Selecciona la rama "main" y la carpeta raíz (/root), guarda.
7. ¡Listo! Te dará una URL tipo: https://tuusuario.github.io/harmonie-web

Para cambiar las imágenes:
- Reemplaza el archivo en /img por uno nuevo con el mismo nombre o cambia el nombre en index.html
